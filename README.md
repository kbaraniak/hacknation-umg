# Stack

## Frontend

Frontend oparty na **React** i **Next.js 15**, zaprojektowany z myślą o wydajności, dostępności i skalowalnym UI.

### Technologie

- **Next.js 15** – SSR, routing, optymalizacja wydajności  
- **React** – komponentowa architektura UI  
- **TypeScript** – silne typowanie i lepsze DX  
- **HeroUI v2** – nowoczesne, dostępne komponenty  
- **Tailwind CSS** – utility-first styling  
- **Tailwind Variants** – skalowalne warianty komponentów  
- **Framer Motion** – animacje i mikrointerakcje  
- **next-themes** – obsługa motywów (light / dark / system)

---

## Backend

Backend zbudowany w **FastAPI**, nastawiony na wysoką wydajność, asynchroniczność i bezpieczeństwo typów.

### Technologie

- **Python** – główny język backendu  
- **FastAPI** – szybkie, typowane API  
- **Uvicorn** – serwer ASGI  
- **pandas** – przetwarzanie i analiza danych  
- **statsmodels** – modelowanie statystyczne  
- **httpx** – asynchroniczna komunikacja HTTP  
- **pytest** – testy jednostkowe i integracyjne

---

## Architektura

- Pełne typowanie frontend ↔ backend  
- Asynchroniczny backend dla wysokiej przepustowości  
- Modularny i skalowalny system UI  
- Animacje jako integralna część UX  
- Backend rozwijany w podejściu test-first
